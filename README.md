# Chatbot Inteligente do IFPI

Projeto prático de Inteligencia Artificial utilizando aprendizado de máquina supervisionado.

## Tecnologias
- Python
- Scikit-learn

## Descrição 

O projeto consiste em um chatbot que classifica perguntas frequentes sobre o IFPI e responde automaticamente com base no padrão aprendido durante o treinamento.
